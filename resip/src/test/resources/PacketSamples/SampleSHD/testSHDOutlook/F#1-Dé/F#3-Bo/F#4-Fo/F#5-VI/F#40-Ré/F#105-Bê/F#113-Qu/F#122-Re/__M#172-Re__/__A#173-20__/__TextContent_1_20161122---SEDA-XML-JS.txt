<StorageRule><Rule>...</StorageRule>
<AppraisalRule><Rule>...</AppraisalRule>

Dans Vitam, on obtient :
StorageRule : { Rule : … },
AppraisalRule : { Rule : … }

La correspondance est donc totale.
