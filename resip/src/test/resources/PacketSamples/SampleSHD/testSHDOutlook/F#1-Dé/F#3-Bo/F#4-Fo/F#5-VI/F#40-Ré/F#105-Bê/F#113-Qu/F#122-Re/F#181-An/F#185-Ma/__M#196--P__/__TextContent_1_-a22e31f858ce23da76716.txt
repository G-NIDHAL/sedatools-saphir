Résumé:                     [Besoin Métier légitime] Impossibilité de
modifier le nom d'un fichier dans VITAM
Description: 
Nous avons tenté de modifier le nom d'un fichier, la balise FileName contenu
dans le bloc BinaryDataObject sans y parvenir.

Il semblerait qu'il soit impossible de procéder a des modification dans la
collection Objects, cependant un archiviste peut vouloir modifier le nom d'un
fichier après avoir insérer son SIP, après s'être rendu compte d'une erreur
par exemple.

Voici la requête utilisée avec l'ID de l'objet dans l'URL: 

{
  "$roots": [],
  "$query": [],
  "$filter": {
    "$orderby": {
      "TransactedDate": 1
    }
  },
  "$action": [
    {
      "$set": {
        "FileInfo.FileName": "Toto"
      }
    }
  ]
}
