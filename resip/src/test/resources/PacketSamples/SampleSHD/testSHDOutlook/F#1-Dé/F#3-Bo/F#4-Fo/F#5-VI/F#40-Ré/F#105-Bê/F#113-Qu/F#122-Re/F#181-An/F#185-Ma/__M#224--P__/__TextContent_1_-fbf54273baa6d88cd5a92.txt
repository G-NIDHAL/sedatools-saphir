Résumé:                     [Besoin Métier Légitime] - Impossibilité de
modifier les MD de type Management MetadataType
Description: 
Ni via l'IHM; ni via l'API, il n'est possible de modifier ces métadonnées.

Les métadonnées contenus dans le manifest, dans le bloc Management
MetadataType se retrouvent dans VITAM soit dans la collection Unit (Rules), soit
dans la collection AccessionRgister

Or il n'est pas possible de modifier le MD concernant les Rules dans la
collection Unit et il n'est pas possible de faire de modification sur la
collection AccessionRegister.
